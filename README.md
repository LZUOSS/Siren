# Siren: OSS 静态站



### 基本结构

* 首页
* News 新闻页
* Projects 项目介绍页
* Mirror 镜像站
* Members 成员页面
* Archive 存档
* .hide 独立页面（纳新，SFD等活动，不在导航栏里展示，但是留下入口）
